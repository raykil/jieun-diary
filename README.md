# 달의 기록 (jieun-diary)

A menstrual cycle tracking app — packaged for iPhone using Capacitor.

## Prerequisites

| Tool | Notes |
|------|-------|
| Node.js 18+ | https://nodejs.org |
| Xcode 15+ | Mac App Store (macOS only) |
| Apple Developer account | Free account works for sideloading |

---

## First-time setup

```bash
# 1. Install dependencies
npm install

# 2. Add the iOS platform (generates the ios/ Xcode project)
npx cap add ios

# 3. Sync the web app into the iOS project
npx cap sync

# 4. Open in Xcode
npx cap open ios
```

Inside Xcode:
1. Connect your iPhone via USB and select it as the build target.
2. Go to **Signing & Capabilities** → select your Apple ID team.
3. Press **▶ Run**.

The app will install directly on your iPhone without going through the App Store.

---

## Ongoing development

All edits happen in `www/index.html`. After saving your changes, run:

```bash
npx cap sync
```

Then build again from Xcode (or hit ⌘R if it's already open).

## Project structure

```
jieun-diary/
├── www/
│   └── index.html        ← edit this
├── capacitor.config.json
├── package.json
└── README.md
```

The `ios/` folder is generated by Capacitor and is git-ignored — re-create it anytime with `npx cap add ios`.
